# Proyecto Web - Catálogo de Productos

## Descripción
Este es un proyecto básico de una página web tipo catálogo, desarrollado como práctica de desarrollo web.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Menú y submenús de navegación
- Carrusel de imágenes
- Sección de productos con descripción breve y precios

## Estado del proyecto
En desarrollo. Publicado, en proceso de habilitacion para acceso público.

## Aprendizaje
Proyecto realizado como práctica personal para mejorar habilidades en desarrollo web y familiarizarme con la estructura de páginas web.
